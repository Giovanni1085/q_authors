The file APS_author2DOI.dat contains the result of the disambiguation procedure of the APS dataset, described in Supplementary Materials SM1.1

Each line of the file corresponds to one of the disambiguated authors, and contains three comma separated fields: 
(i) a numeric id associated to the author 
(ii) name of the author, 
(iii)a list of tab separated paper DOIs published by the author. The papers are NOT chronologically ordered
